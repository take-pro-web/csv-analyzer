# 📊 CSV自動分析ツール

CSVファイルをアップロードするだけで、グラフ・統計を自動で表示するWebアプリです。

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Streamlit](https://img.shields.io/badge/Streamlit-1.x-red)

---

## 🎯 こんな方に

- Excelでの集計・グラフ作成を自動化したい
- 売上データを手軽に可視化したい
- 非エンジニアでも使えるデータ分析ツールが欲しい

---

## ✨ 機能

- CSVをアップロードするだけで即座に分析
- 折れ線グラフ・棒グラフを切り替え表示
- 平均・中央値・標準偏差などの統計サマリーを自動計算
- データテーブル表示

---

## 🚀 使い方

### インストール
pip install streamlit pandas matplotlib

### 起動
streamlit run app.py

ブラウザが自動で開きます。
